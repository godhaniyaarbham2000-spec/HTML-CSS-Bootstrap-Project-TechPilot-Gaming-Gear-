# 🎮 TechPilot - Gaming Gear & Accessories Website

A responsive **HTML + CSS + Bootstrap web project** featuring a modern gaming gear landing page with sections like banner, products, services, stats, and contact.  
Built for practice and portfolio showcase using frontend technologies.

--- 

## 🚀 Features

* 🎨 Modern gaming UI design  
* 📱 Fully responsive layout (Bootstrap grid system)  
* 🛒 Product showcase section  
* 📊 Stats / Facts section  
* 🧩 Services section with icons  
* 📩 Contact + Newsletter form UI  
* 🎯 Smooth navigation with scroll effects  
* 🔗 Font Awesome icons integration  

---

## 🛠️ Tech Stack

* HTML5  
* CSS3  
* Bootstrap 5  
* Font Awesome Icons  
* Google Fonts
